# Data

This folder contains representative sample datasets used in the study.

The coordinate CSV files contain labeled spatial samples used for training and testing the machine learning classifiers for socioeconomic class prediction.

The socioeconomic labels were derived using secondary data obtained from:

- Department of Stamps and Registration, Karnataka  
  https://kaveri.karnataka.gov.in/landing-page

- Directorate of Registration and Stamp Revenue, Finance Department, Government of West Bengal  
  https://wbregistration.gov.in/MV/mv_aprt.aspx

For this study, property guidance values were categorized into income-based socioeconomic classes for Bangalore and Kolkata.

These files are provided to support reproducibility of the workflow presented in the manuscript.
