# Notebooks

This folder contains Jupyter notebooks used for preprocessing, exploratory analysis, and classification workflows.

## Contents

1. `01_prisma_he5_to_tiff.ipynb`
   - Converts PRISMA hyperspectral `.he5` files into GeoTIFF format.

2. `02_pca_boxplot_analysis.ipynb`
   - Performs PCA analysis, scree plot generation, and socioeconomic class separability analysis.

3. `03_classification_workflow.ipynb`
   - Performs machine learning classification using PRISMA and Landsat spectral features.
